Changelog

Every arcana release — features, fixes, and infrastructure updates. Published from CI on every tagged push.

v0.3.44 — latest Jul 22, 2026
  • Session status breakdown chart — stacked horizontal bar with clickable legend to filter by status.
  • "Load more" pagination on Sessions page — fetches older sessions in batches of 200 via cursor-based pagination.
  • Dashboard usage card now reads daily limit from proxy (no more hardcoded 50) — shows "Daily limit unknown" or "Unlimited" for enterprise.
  • Dashboard refresh button with rotate animation — reloads all data on demand.
  • Account tier badge in sidebar across all workspace pages — fetched from /v1/health on every page load.
  • Sidebar usage bar — mini daily request usage bar with color thresholds (green/yellow/red).
  • Sidebar tier tooltip — hover to see daily used/limit/remaining info from /v1/usage.
  • Dashboard heading tier badge — shows account tier (Free/Pro/Enterprise) next to the page title.
  • "Last refreshed" timestamp on dashboard — updates on every successful data load.
  • Status badge tooltips on session rows — hover for description ("Finished normally", "Streamed over SSE", "Error occurred").
  • Console logging of session status counts on load — helps debug what the proxy returns.
  • Increased session fetch limit from 50 to 200 across Sessions and Dashboard pages.
  • Dashboard daily usage no longer hardcodes 50 as fallback limit — uses server-provided value or shows "—".

  • JSON-LD structured data (TechArticle + BreadcrumbList) on all documentation pages for SEO.
  • Version badge (v0.3.15) added to the homepage hero section.

  • Improved search UX with slide-in animations, gradient highlights, and keyboard accessibility.
  • Enhanced sidebar navigation with gradient hover states, glowing active indicators, and smooth transitions.
  • Smooth scroll behavior and better anchor link transitions with highlight animation.
