Cron Jobs

Schedule recurring prompts, reports, reviews, and checks from the same Arcana runtime.

Create a Job

# Daily standup report
arcana cron add --schedule "0 9 * * *" --prompt "Summarize yesterday's commits"

# Weekly code review
arcana cron add --schedule "0 17 * * 5" --prompt "Review open PRs"

Manage Jobs

# List all jobs
arcana cron list

# Run a job immediately
arcana cron run <job-id>

# Remove a job
arcana cron remove <job-id>
