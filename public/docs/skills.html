Skills

Skills are reusable, self-contained instruction sets that you can load on demand to give Arcana specialized knowledge or behavior. Instead of cluttering every session with every possible instruction, skills let you load expertise only when you need it — keeping your base context clean and your responses focused.

How Skills Work

A skill is a set of instructions (like a mini-prompt) stored in a local file. When you load a skill, its instructions are injected into the current session context, temporarily teaching the AI new capabilities, constraints, or domain knowledge. Skills persist between sessions but are only active when explicitly loaded.

Finding Skills

Arcana includes several built-in skills and a growing library of community-contributed skills:

# List all locally available skills
arcana skills list

# Search the community skill registry
arcana skills search "database"
arcana skills search "testing"

# View details about a skill
arcana skills info arcana-mcp-fs

Loading a Skill

Load a skill during an active session. The skill's instructions are added to the AI's context:

# Load a skill by name
arcana skills load arcana-mcp-fs

# Load multiple skills at once
arcana skills load arcana-mcp-fs arcana-git-workflow

# Load a skill from a file path
arcana skills load ./path/to/skill.md

# List currently loaded skills
arcana skills loaded

Creating a Skill

Skills are plain Markdown files with a YAML frontmatter header. Create one in the ~/.config/arcana/skills/ directory:

---
name: react-expert
description: Expert-level React and Next.js guidance
version: 1.0.0
tags: [react, nextjs, frontend]
---

You are a React expert with deep knowledge of:
- React 19 features and server components
- Next.js App Router patterns
- State management with Zustand and TanStack Query
- Performance optimization and bundle splitting

When answering React questions:
1. Prefer server components by default
2. Use TypeScript in all examples
3. Include error boundaries for async operations
4. Show both client and server component patterns

Then use the CLI to register it:

# Create a new skill interactively
arcana skills create react-expert

# Edit an existing skill
arcana skills edit react-expert

# Export a skill to share
arcana skills export react-expert --file react-expert.md

Community Skills

Install community-contributed skills from GitHub repositories. These are vetted instruction sets shared by the Arcana community:

# Search the community registry
npx skills find database
npx skills find testing

# Install a community skill repo
npx skills add owner/repo --list
npx skills add owner/repo --skill skill-name --yes

Tips for Effective Skills

  • Be specific — Narrow instructions produce more reliable results
  • Use examples — Include code snippets showing the preferred patterns
  • Set constraints — Tell the AI what not to do
  • Keep them focused — A skill should do one thing well
  • Version your skills — The version field helps track changes
