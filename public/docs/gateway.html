Chat Gateways

Route your Arcana sessions to external chat platforms. When the terminal is not the right surface — or you want to give teammates access — gateways let you interact with Arcana from Telegram, Discord, Slack, or WhatsApp.

How Gateways Work

A gateway connects Arcana to a chat platform as a bot or webhook integration. Once configured, you can route a session (or all new sessions) to that platform, allowing you to send prompts and receive responses directly in your chat app. Arcana handles authentication, rate limiting, and message formatting automatically.

Supported Platforms

PlatformIntegration TypeFeatures
TelegramBot APIInline queries, group chats, commands
DiscordBot or WebhookSlash commands, threads, channels
SlackApp or WebhookChannels, DMs, shortcuts
WhatsAppBusiness APIDirect messages, group chats

Telegram Setup

Create a bot via @BotFather on Telegram, then add it to Arcana:

# Add the Telegram gateway with your bot token
arcana gateway add telegram --token 123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11

# List active gateways
arcana gateway list

# Route the current session to Telegram
arcana gateway route telegram

Discord Setup

# Add a Discord webhook gateway
arcana gateway add discord --webhook https://discord.com/api/webhooks/...

# Or add a Discord bot gateway (requires bot token)
arcana gateway add discord --token BOT_TOKEN --guild GUILD_ID

Slack Setup

# Add a Slack webhook gateway
arcana gateway add slack --webhook https://hooks.slack.com/services/...

# Or add a Slack app
arcana gateway add slack --token xoxb-... --channel general

WhatsApp Setup

# Add a WhatsApp Business API gateway
arcana gateway add whatsapp --phone-id PHONE_ID --token ACCESS_TOKEN

Managing Gateways

# List all configured gateways
arcana gateway list

# Remove a gateway
arcana gateway remove telegram

# Test a gateway connection
arcana gateway test discord

# Set a default gateway for all sessions
arcana config set gateway.default telegram
