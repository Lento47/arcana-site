Chat Gateways

Route your Arcana sessions to external chat platforms. When the terminal is not the right surface, use a gateway.

Supported Platforms

  • Telegram — Bot API integration
  • Discord — Bot or webhook integration
  • Slack — App or webhook integration
  • WhatsApp — Business API integration

Setup

# Add a Telegram gateway
arcana gateway add telegram --token BOT_TOKEN

# List gateways
arcana gateway list

# Route a session
arcana gateway route --session <id> --gateway telegram
