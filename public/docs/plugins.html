Plugins

Extend Arcana's functionality with plugins. Plugins can add new tools, modify behavior, or integrate with external services.

Install a Plugin

# Install from registry
arcana plugin install @arcana/git-tools

# Install from URL
arcana plugin install https://github.com/user/plugin

Manage Plugins

# List installed plugins
arcana plugin list

# Update plugins
arcana plugin update --all

# Remove a plugin
arcana plugin remove @arcana/git-tools

Create a Plugin

Plugins follow a standard interface. See the Plugin SDK for details.
