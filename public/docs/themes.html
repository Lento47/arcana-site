Themes

Arcana includes 7 built-in arcane themes with custom background images for supported truecolor terminals.

Available Themes

  • default — Clean dark theme
  • bloodmoon — Deep red accents
  • coven — Purple and violet tones
  • crypt — Stone gray and gold
  • dragon — Fiery orange and amber
  • lich — Ghostly green and teal
  • wraith — Shadowy blue and silver

Switch Themes

# Set theme
arcana config set theme bloodmoon

# List available themes
arcana theme list
