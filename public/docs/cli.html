CLI Commands

Complete reference for all Arcana CLI commands. Run arcana --help for a quick summary.

Core Commands

arcana

Start an interactive session. This is the default command when no arguments are provided.

arcana

arcana serve

Start the Arcana server with HTTP API support.

arcana serve [--port 3000]

Configuration

arcana config

View or set configuration values.

# View all config
arcana config --list

# Set a value
arcana config set <key> <value>

# Get a value
arcana config get <key>

Key Management

arcana key

Manage provider API keys.

# Add a key
arcana key add <provider> <api-key>

# List keys
arcana key list

# Remove a key
arcana key remove <provider>

Memory

arcana memory

Manage session memory and facts.

# Search memory
arcana memory search <query>

# List facts
arcana memory facts

# Clear memory
arcana memory clear

Tools

arcana tools

List and manage available tools.

# List all tools
arcana tools list

# Check tool permissions
arcana tools permissions

Plugins

arcana plugin

Install, list, and manage plugins.

# Install a plugin
arcana plugin install <name>

# List installed plugins
arcana plugin list

# Remove a plugin
arcana plugin remove <name>

Credits

arcana credits

Manage proxy credits for pay-as-you-go routing.

# Check balance
arcana credits balance

# Buy credits
arcana credits buy <amount>

License

arcana license

Manage your Arcana license.

# Activate license
arcana license activate <key>

# Check status
arcana license status
