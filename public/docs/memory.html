Memory

Arcana's wiki-style memory system uses FTS5-powered search, extracted facts, and optional cloud sync across machines.

How It Works

During each session, Arcana automatically extracts facts and stores them in a local SQLite database with full-text search.

Search

# Search memory
arcana memory search "project architecture"

# List all facts
arcana memory facts

# Filter by topic
arcana memory facts --topic "api"

Cloud Sync

Enable cloud sync to share knowledge across machines:

# Enable cloud sync
arcana config set memory.cloudSync true

# Push local facts
arcana memory push

# Pull from cloud
arcana memory pull
