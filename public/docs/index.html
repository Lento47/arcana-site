ARCANA Documentation

Welcome to the official documentation for ARCANA — the terminal interface for AI keys. This guide covers installation, configuration, and advanced features.

Quick Start

Install Arcana from npm and start using your AI provider keys immediately:

# Install and run
npx arcana-ai@latest

# Or install globally
npm install -g arcana-ai
arcana

What is Arcana?

Arcana is a terminal-native AI workbench that lets you use multiple AI model providers (OpenAI, Anthropic, DeepSeek, OpenRouter, Cerebras, Ollama, and more) from a single interface. It adds memory, tools, and scheduled runs on top of your existing API keys.

Features

  • Multi-provider support — Use any model key from one terminal
  • Wiki-style memory — FTS5-powered session memory with extracted facts
  • Scoped tools — Shell, filesystem, git, and skill tools with explicit permissions
  • Skills on demand — Load specialized instructions when needed
  • Cron jobs — Schedule recurring prompts and reports
  • Chat gateways — Route sessions to Telegram, Discord, Slack, or WhatsApp
  • Themes — 7 arcane themes with custom backgrounds
  • Cloud sync — Push local facts to the cloud and share across machines

Architecture

Arcana sits one layer above your model provider. It provides a session layer with:

  • Key management — Add provider keys once, switch models by task
  • Tool profiles — Explicit permissions for shell, filesystem, and gateway access
  • Memory system — Extracted facts, full-text search, and optional cloud sync
  • Job scheduler — Recurring prompts, reviews, and checks from the same runtime
