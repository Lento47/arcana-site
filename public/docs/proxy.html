Proxy API

Route AI model requests through Arcana's proxy. Use your own provider keys for direct billing, or buy Arcana proxy credits for pay-as-you-go routing.

Endpoints

# Chat completions
POST https://proxy.arcana.ai/v1/chat/completions

# Models list
GET https://proxy.arcana.ai/v1/models

# Credits balance
GET https://proxy.arcana.ai/v1/credits/balance

Authentication

# Use your Arcana API key
Authorization: Bearer arcana-...

# Or use provider key directly
Authorization: Bearer sk-...

Credits

Proxy credits cost $1 = 100 credits, minimum $5. Buy credits via the web dashboard or CLI:

arcana credits buy 500
