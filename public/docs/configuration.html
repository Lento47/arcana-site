Configuration

Arcana is configured through a combination of arcana.json in your project root and environment variables.

Config File

Create an arcana.json file in your project root:

{
  "defaultModel": "openai/gpt-4",
  "providers": {
    "openai": { "key": "sk-..." },
    "anthropic": { "key": "sk-ant-..." }
  },
  "tools": {
    "shell": { "enabled": true },
    "filesystem": { "enabled": true, "root": "." }
  },
  "memory": { "enabled": true, "cloudSync": false },
  "theme": "bloodmoon"
}

Environment Variables

# Provider keys
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=sk-ant-...
DEEPSEEK_API_KEY=...
OPENROUTER_API_KEY=sk-or-...

# Arcana settings
ARCANA_THEME=bloodmoon
ARCANA_PROXY_URL=https://proxy.arcana.ai

Themes

Arcana includes 7 built-in themes: bloodmoon, coven, crypt, dragon, lich, wraith, and default.
