Troubleshooting

Common issues and solutions for Arcana.

Installation Issues

npx arcana-ai fails

Ensure you have Node.js 18+ installed:

node --version  # Should be 18+
npm install -g arcana-ai

Provider Key Issues

"Invalid API key" error

Verify your key is correct and has the right permissions:

arcana key list
arcana key add openai sk-your-key

Memory Issues

Memory search returns empty

Ensure memory is enabled and facts have been extracted:

arcana config get memory.enabled
arcana memory facts
